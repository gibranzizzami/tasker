# Install di ubuntu

### Update sistem ubuntu

```
sudo apt update
```

### Install Nodejs dan package managernya

```
sudo apt install nodejs npm build-essential
```

### Install terminalizernya

```
sudo npm install -g terminalizer
```

### Cek versi

```
terminalizer --version
```
