# Dokumentasi Pembuatan Akun GitHub Menggunakan Google Account

---

## 1. Langkah-Langkah Pembuatan Akun

### Step 1: Buka Website GitHub
1.  Buka browser web (Chrome, Firefox, Edge, dll.)
2.  Kunjungi alamat: **[https://github.com/](https://github.com/)**
3.  Klik tombol **"Sign up"** yang berada di pojok kanan atas halaman.

<img width="2880" height="1920" alt="image" src="https://github.com/user-attachments/assets/40c8f416-63bd-4bf0-b7c7-e2411408a43d" />


### Step 2: Pilih Metode Registrasi dengan Google
1.  Pada halaman registrasi, Anda akan melihat beberapa opsi:
    - **Email and password** (metode tradisional)
    - **Continue with Google**
    - **Continue with Apple**

2.  **Klik tombol "Continue with Google"**

![Uploading image.pngâ€¦]()

### Step 3: Pilih Akun Google
1.  Browser akan membuka pop-up atau mengarahkan Anda ke halaman login Google.
2.  **Pilih akun Google** yang ingin Anda gunakan untuk mendaftar GitHub.
3.  Jika sudah login ke Google, proses akan otomatis melanjutkan.
4.  Jika belum login, masukkan email dan password Google Anda.

### Step 4: Izinkan Akses ke GitHub
1.  Google akan meminta persetujuan untuk memberikan akses ke GitHub.
2.  **Baca persyaratan** yang diminta.
3.  **Klik "Continue"** atau **"Allow"** untuk memberikan izin.


---

## 3. Verifikasi Email

### Step 5: Verifikasi Alamat Email (Opsional)
1.  Meskipun menggunakan Google Account, GitHub mungkin masih meminta verifikasi email.
2.  Periksa inbox email Google Anda yang terkait.
3.  Cari email dari GitHub dengan subjek **"Verify your email address"**.
4.  **Klik tombol verifikasi** di dalam email tersebut.
