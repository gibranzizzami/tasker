# Pembuatan Akun GitHub 
---
Berikut adalah cara pembuatan akun GitHub 

1. Buka Firefox Browser
2. Ketik `github account` pada searchbar dan cari
3. Pilih laman **Sign up for GitHUb**
4. Masukkan email, password, username, dan negara
![ISi](https://github.com/zihannrae/tasker/blob/73ea7a851fec2b8352dc5dd22381f3fc784be2c5/12402051030056_zihan_nuraeni/tugas/session-1/img/github%20acc.jpeg)
5. KLik **Create Account**
![create acc](https://github.com/zihannrae/tasker/blob/73ea7a851fec2b8352dc5dd22381f3fc784be2c5/12402051030056_zihan_nuraeni/tugas/session-1/img/create%20acc.jpeg)
6. Lakukan captcha 
7. Masukkan code yang dikirim oleh GitHub melalui Email
8. Akun GitHub sudah bisa digunakan
