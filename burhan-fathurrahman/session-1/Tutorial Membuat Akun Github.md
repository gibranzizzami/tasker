Tutorial Membuat akun Github

Buka Browser yang biasa Anda gunakan seperti Mozilla Firefox, Google Chrome atau Microsoft Edge

Lalu ketik Github atau bisa menggunakan link github.com

Pilih kolom Sign Up, lalu masukkan data yang dibutuhkan seperti email anda yang aktif, password, username, country/region, dan terakhir klik create account

Membuat akun github juga bisa dengan menggunakan akun google Anda. Cara nya klik "Continue with Google", lalu pilih akun google yang ingin Anda gunakan. Lalu kolom email akan terisi otomatis sesuai dengan email anda, lalu anda hanya perlu memasukkan username dan memilih contry/region, dan terakhir anda klik "Create accout"
